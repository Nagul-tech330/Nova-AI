## 📖 About

Nova AI is an intelligent desktop voice assistant developed using Python. It can recognize voice commands, speak responses, automate desktop tasks, search the web, and provide useful daily assistance through a modern graphical interface.

---

## ✨ Features

- 🎤 Voice Recognition
- 🔊 Text-to-Speech
- 🌐 Browser Search
- 📸 Screenshot Capture
- 📰 News Module
- 🌦 Weather Module
- 🧠 AI Chat Module
- 💻 Desktop Automation
- 🎨 Modern GUI
- 📂 Modular Project Structure

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Nagul-tech330/Nova-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
